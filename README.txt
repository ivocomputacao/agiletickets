Para pegar as bibliotecas faltantes e ter o projeto completamente configurado, no terminal, rode:

> ant compile

Pode demorar, na primeira vez, porque o Ivy baixará as dependências desse projeto, mas nas próximas
deve ser bastante rápido.

TDD (baby steps)
- Testa tudo
- Faz pensar no teste/requisito
- O teste não é viciado
- Faz somente o necessário, não cria um framework para atender a lógica!
- Segurança para fazer refatoração de código.
- Pensar no proejto de classes.
- No baby steps, fazer o baby step quando o código está confuso, se estiver claro, manda bala!
- iniciar com as amostra mais simples para ajudar no entendimento!
